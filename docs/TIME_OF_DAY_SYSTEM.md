# 🎨 Единая система визуальных стилей времени суток

## 📋 Обзор

Это ЕДИНАЯ система визуальных параметров, которая обеспечивает согласованность между:
- **Three.js сценой** (освещение, небо, облака)
- **CSS/React стилями** (фоны, градиенты, цвета)  
- **UI компонентами** (текст, оверлеи)

**Философия**: Не создавать разные стили для каждого случая, а использовать единые параметры (токены/пресеты), которые плавно меняются по времени суток.

---

## 🌅 Визуальные параметры по времени суток

### 1. **РАССВЕТ** (Sunrise) — 5:00-7:00

**Общая атмосфера**: Теплая, пастельная, мягкая, романтичная

| Параметр | Значение | Описание |
|----------|----------|----------|
| **Небо** | `#FF6B6B → #FFB347 → #FFE4B5` | Коралловый → оранжево-персиковый → кремовый |
| **Облака** | `#FFB6C1` (розовые) | Полупрозрачные (60%), мягкие края |
| **Освещение** | Теплое, мягкое | Ambient: 0.5, Directional: 0.6 золотисто-оранжевый |
| **Туман** | Умеренный (0.015) | Розоватый `#FFE4E1` |
| **Насыщенность** | 80% | Высокая, но не перегруженная |
| **Контраст** | 60% | Низкий, мягкие переходы |
| **Теплота** | 90% | Очень теплые тона |

**Применение**:
- Солнце низко слева, свечение 1.2x
- Земля еще в тени (темно-серо-синяя)
- Видимость слегка размыта (75%)

---

### 2. **ДЕНЬ** (Day) — 7:00-17:00

**Общая атмосфера**: Яркая, чистая, четкая, энергичная

| Параметр | Значение | Описание |
|----------|----------|----------|
| **Небо** | `#1E88E5 → #42A5F5 → #90CAF9` | Насыщенный синий → голубой → светло-голубой |
| **Облака** | `#FFFFFF` (белые) | Непрозрачные (80%), четкие края |
| **Освещение** | Яркое, нейтральное | Ambient: 0.7, Directional: 1.0 белый |
| **Туман** | Минимальный (0.005) | Светло-голубой `#E3F2FD` |
| **Насыщенность** | 100% | Максимальная |
| **Контраст** | 100% | Четкие границы |
| **Теплота** | 50% | Нейтральная |

**Применение**:
- Солнце высоко в зените, свечение 1.5x
- Идеальная видимость (100%)
- Земля освещена (зеленоватая)

---

### 3. **ЗАКАТ** (Sunset) — 17:00-19:00

**Общая атмосфера**: Драматичная, теплая, насыщенная, кинематографичная

| Параметр | Значение | Описание |
|----------|----------|----------|
| **Небо** | `#FF6347 → #FF8C42 → #FFD700` | Томатно-красный → оранжево-золотой → золотой |
| **Облака** | `#FF7F50` (коралловые) | Полупрозрачные (75%), золотые блики |
| **Освещение** | Теплое, драматичное | Ambient: 0.45, Directional: 0.7 красно-оранжевый |
| **Туман** | Умеренный (0.012) | Бисквитный `#FFE4C4` |
| **Насыщенность** | 95% | Очень высокая |
| **Контраст** | 75% | Средний |
| **Теплота** | 100% | Максимально теплые тона |

**Применение**:
- Солнце низко справа, свечение 1.3x
- Видимость хорошая (80%)
- Земля начинает темнеть (серо-фиолетовая)

---

### 4. **НОЧЬ** (Night) — 19:00-5:00

**Общая атмосфера**: Холодная, темная, таинственная, спокойная

| Параметр | Значение | Описание |
|----------|----------|----------|
| **Небо** | `#0B1026 → #1A2238 → #2C3E50` | Почти черный → темно-синий → синевато-серый |
| **Облака** | `#4A5568` (темно-серые) | Полупрозрачные (40%), очень мягкие края |
| **Освещение** | Холодное, приглушенное | Ambient: 0.25, Directional: 0.35 лунный голубовато-серый |
| **Туман** | Легкий (0.008) | Темно-синий `#2C3E50` |
| **Насыщенность** | 50% | Приглушенные цвета |
| **Контраст** | 50% | Низкий |
| **Теплота** | 20% | Холодные тона |

**Применение**:
- Луна высоко, свечение 0.8x (нет солнца)
- Ограниченная видимость (60%)
- Земля очень темная
- Звезды видны (1200 звезд)

## 🔧 Использование в коде

### **1. Three.js сцена (Освещение)**

```javascript
import { getLightingParams } from '../utils/timeOfDayVisuals';

function MyThreeScene({ timeOfDay }) {
  const lighting = getLightingParams(timeOfDay);
  
  return (
    <>
      <ambientLight 
        intensity={lighting.ambient.intensity}
        color={lighting.ambient.color}
      />
      <directionalLight
        position={lighting.directional.position}
        intensity={lighting.directional.intensity}
        color={lighting.directional.color}
        castShadow
      />
    </>
  );
}
```

### **2. Three.js облака**

```javascript
import { getCloudParams } from '../utils/timeOfDayVisuals';

function CloudScene({ timeOfDay }) {
  const cloudParams = getCloudParams(timeOfDay);
  
  return (
    <Cloud
      opacity={cloudParams.opacity}
      color={cloudParams.color}
      // ... другие параметры
    />
  );
}
```

### **3. CSS/React фоновый градиент**

```javascript
import { getCSSVars } from '../utils/timeOfDayVisuals';

function WeatherBackground({ timeOfDay }) {
  const cssVars = getCSSVars(timeOfDay);
  
  return (
    <div style={{
      background: cssVars['--sky-gradient'],
      color: cssVars['--text-primary']
    }}>
      {/* Контент */}
    </div>
  );
}
```

### **4. Применение CSS переменных глобально**

```javascript
import { applyCSSVars } from '../utils/timeOfDayVisuals';

useEffect(() => {
  applyCSSVars(document.documentElement, timeOfDay);
}, [timeOfDay]);
```

### **5. Определение времени суток по часу**

```javascript
import { getTimeOfDayFromHour } from '../utils/timeOfDayVisuals';

const currentHour = new Date().getHours();
const timeOfDay = getTimeOfDayFromHour(currentHour); // 'sunrise' | 'day' | 'sunset' | 'night'
```

### **6. Плавные переходы (интерполяция)**

```javascript
import { interpolateTimeOfDay } from '../utils/timeOfDayVisuals';

// Плавный переход от дня к закату
const visuals = interpolateTimeOfDay('day', 'sunset', 0.5); // 50% прогресс
```

## 🌦️ Модификаторы погоды

Система также поддерживает модификаторы для разных погодных условий:

### **Облачная погода**

```javascript
import { getTimeOfDayVisuals, applyCloudyModifier } from '../utils/timeOfDayVisuals';

const baseVisuals = getTimeOfDayVisuals('day');
const cloudyVisuals = applyCloudyModifier(baseVisuals);

// Результат:
// - Освещение на 30% темнее
// - Облаков на 50% больше
// - Видимость на 20% хуже
```

### **Дождливая погода**

```javascript
const rainyVisuals = applyRainyModifier(baseVisuals);

// Результат:
// - Освещение на 50-70% темнее
// - Туман в 2 раза плотнее
// - Насыщенность -30%
// - Контраст -20%
```

### **Туман/мгла**

```javascript
const mistyVisuals = applyMistyModifier(baseVisuals);

// Результат:
// - Туман в 4 раза плотнее
// - Видимость 30%
// - Контраст -50%
// - Насыщенность -40%
```

## ✅ Преимущества единой системы

1. **Консистентность**: Один источник правды для всех визуальных параметров
2. **Масштабируемость**: Легко добавить новое время суток (например, "golden hour")
3. **Поддерживаемость**: Все параметры в одном месте
4. **Производительность**: Пресеты вычисляются один раз
5. **Гибкость**: Модификаторы для разных погодных условий
6. **DX**: Простое API для разработчиков

## 🎯 Применение в компонентах

### Рекомендуемая структура

```javascript
// 1. Определить время суток
const timeOfDay = getTimeOfDayFromHour(currentHour);

// 2. Получить базовые визуальные параметры
let visuals = getTimeOfDayVisuals(timeOfDay);

// 3. Применить модификаторы погоды (если нужно)
if (isRaining) visuals = applyRainyModifier(visuals);
if (isCloudy) visuals = applyCloudyModifier(visuals);
if (isMisty) visuals = applyMistyModifier(visuals);

// 4. Использовать параметры в компонентах
const lightingParams = visuals.lighting;
const cloudParams = visuals.clouds;
const cssVars = visuals.cssVars;
```
