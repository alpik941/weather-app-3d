## 🌦️ Модификаторы погоды (применяются поверх базовых параметров)

### Облачная погода (Cloudy)

| Параметр | Модификатор | Результат |
|----------|-------------|-----------|
| Ambient Intensity | ×0.7 | На 30% темнее |
| Directional Intensity | ×0.5 | Свет более рассеянный |
| Плотность облаков | ×1.5 | На 50% больше облаков |
| Прозрачность облаков | ×1.2 (макс 0.9) | Более плотные |
| Видимость | ×0.8 | На 20% хуже |

### Дождливая погода (Rainy)

| Параметр | Модификатор | Результат |
|----------|-------------|-----------|
| Ambient Intensity | ×0.5 | На 50% темнее |
| Directional Intensity | ×0.3 | На 70% темнее |
| Плотность тумана | ×2.0 | В 2 раза плотнее |
| Видимость | ×0.6 | На 40% хуже |
| Насыщенность | ×0.7 | Цвета приглушены |
| Контраст | ×0.8 | Меньше контраста |

### Туман/Мгла (Misty)

| Параметр | Модификатор | Результат |
|----------|-------------|-----------|
| Ambient Intensity | ×0.6 | На 40% темнее |
| Плотность тумана | ×4.0 | В 4 раза плотнее |
| Видимость | = 30% | Фиксированная низкая видимость |
| Дымка | = 80% | Очень высокая дымка |
| Контраст | ×0.5 | На 50% меньше |
| Насыщенность | ×0.6 | На 40% меньше |

## 💾 Использование в коде

### Quick Start

```javascript
import { 
  getTimeOfDayVisuals, 
  getLightingParams, 
  getSkyColors,
  getCloudParams,
  getCSSVars
} from '@/utils/timeOfDayVisuals';

// Получить все параметры сразу
const visuals = getTimeOfDayVisuals('sunset');

// Или получить конкретные части
const lighting = getLightingParams('sunset');
const skyColors = getSkyColors('sunset');
const cloudParams = getCloudParams('sunset');
const cssVars = getCSSVars('sunset');

// С модификаторами погоды
import { applyRainyModifier } from '@/utils/timeOfDayVisuals';
const rainyVisuals = applyRainyModifier(visuals);
```
